APT-MIRROR

Para utilizar este mirror, você deverá abrir o arquivo /etc/apt/source.list com o editor de texto de sua preferência e adicionar os seguintes endereços de repositórios:

# Debian Repository - Lenny
deb http://192.168.69.69/debian/ lenny main contrib non-free
deb-src http://192.168.69.69/debian/ lenny main contrib non-free

# Debian Repository - Lenny Security
deb http://192.168.69.69/debian-security lenny/updates main contrib non-free
deb-src http://192.168.69.69/debian-security lenny/updates main contrib non-free

obs: 192.168.69.69 é o ip do servidor mirror
